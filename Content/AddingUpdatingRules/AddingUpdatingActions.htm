Adding or Updating Actions

In , you can assign Actions to any of the menu items. Adding Actions allows you to perform an action when the item in the is selected.

To add an Action, do the following:

  2. Choose Edit Shortcut. The dialog is displayed.
  3. Click the + next to the Actions button. A new rule is added under the Run a Rule... list.
  4. Choose an action for the rule from the list. The default Action is Run a Rule... The other options available from the drop-down list are:

  5. Run a rule on all rows - runs the rule on all the rows displayed.

  6. Run a command - run a specific command (enter the command in the Command field when this option is selected).

  7. Run a menu item - select a menu item to run from the drop-down list when this option is selected.

  8. Send a NetCommand - run a specific command (enter the command in the Command field when this option is selected).

  9. Open a Custom Dialog - enter the name and title of the custom dialog to be displayed.

  10. Go to a (specific) column - select a specific column from the list of available ones in the Column dialog when this option is selected.

  11. Group by Columns - choose the columns to group the view by selecting them from the list of available columns. [[THIS SHOULD NOT ALLOW A USER TO SELECT THE SAME COLUMN REPEATEDLY. IS THERE A LIMIT TO THE NUMBER OF COLUMNS A USER CAN SELECT? ]]

  12. Ungroup - removes any grouping that exist in the current view.

  13. Open (a) Send order Ticket - select the following fields:

    • Destination
    • Price Option
    • Quantity and the type of quantity
    • Enter the Side for the ticket

  14. Show (a) Chart - enter a name and arguments for the chart to display.

  15. Browse (to a) URL - enter a URL (web address) of a website to display when this shortcut is clicked.
