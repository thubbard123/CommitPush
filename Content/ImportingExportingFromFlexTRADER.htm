Importing a file

To import a  file, do the following:

  2. Choose the menu item. The following items are displayed for Importing a QBXML file (note that the first set of options relate to importing/exporting the entire configuration file and the second set of options relate to the importing/exporting just the configuration file).
  • Import () from Clipboard - choosing this option allows you import a configuration file from your clipboard for an entire .
  • Export ( configuration) to File... - choosing this option allows you to export the configuration file to your machine for an entire .
  • Export (Quick) to Clipboard - choosing this option allows you to save your configuration file to your clipboard for your entire .
  • Import tab from File... - choosing this option allows you to import the configuration file for just the .
  • Import Tab from Clipboard - choosing this option allows you to import the configuration file for just the .
  • Export Tab to File... - choosing this option allows you to export the configuration file to your machine for just the .
  • Export Tab to Clipboard - choosing this option allows you export the configuration to your clipboard for just the .
